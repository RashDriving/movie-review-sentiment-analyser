#  Movie Review Sentiment Analyzer

 ## Overview
This project is a simple **NLP-based sentiment analysis system** that classifies movie reviews as **Positive** or **Negative**.

The application uses **TF-IDF vectorization** and **Logistic Regression** for text classification and is deployed using **Streamlit** for interactive usage.



 ## Features
Classifies movie reviews into positive or negative sentiment
Uses NLP techniques for text feature extraction
Interactive Streamlit web interface
Lightweight and easy to run



##  Technologies Used
Python
Pandas
Scikit-learn

NLP (TF-IDF)
